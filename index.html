Welcome to HTML Basics

HTML stands for HyperText Markup Language. It is the standard language used to create webpages.

Basic HTML Structure

Every HTML document has a basic structure like this:

<!DOCTYPE html>
<html>
  <head>
    <title>Your Page Title</title>
  </head>
  <body>
    <h1>This is a heading</h1>
    <p>This is a paragraph.</p>
  </body>
</html>

HTML Elements

HTML elements are the building blocks of HTML pages. Examples of common elements:

  • <p> - Defines a paragraph.
  • <h1> to <h6> - Define headings, with <h1> being the most important.
  • <a> - Defines a hyperlink.
  • <img> - Embeds an image.

Why HTML is Important

HTML forms the foundation of the web. Without HTML, there would be no way to structure content for webpages. By understanding HTML, you can create your own websites and better understand how the web works!
